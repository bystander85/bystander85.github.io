tracert用法

发表于 2022-05-20 更新于 2022-05-24 分类于 cmd 阅读次数：

tracert是一个路由跟踪命令，用于确定 IP 数据包访问目标所采取的网络路径，在大型网络排查问题中起到一定的作用。

tracert命令常用在哪里？
利用tracert命令，我们可以查看网络中的走向也可以查看网络经过的节点与节点之间存在的延迟。在大型网络中存在很多节点设备来连接不同的上网设置。错综复杂的网络我们就需要用到tracert来分析网络中的问题。
