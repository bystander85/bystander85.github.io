文字

  • 加粗
    需要加粗的文字部分左右用两个*号包裹起来
  • 斜体
    需要倾斜的文字部分左右用一个*号包裹起来
  • 斜体加粗
    三个*号
  • 中划线
    需要中划线的文字部分左右用两个~号包裹起来
  • 下划线
    不推荐使用下划线，容易和超链接混淆，非得用的话，使用html语法
  • 文字颜色
    使用html的font实现
  • 文字高亮
    需要高亮的文字部分左右用两个=号包裹起来
  • 文字背景色
    借助 table, tr, td 等表格标签的 bgcolor 属性来实现背景色
